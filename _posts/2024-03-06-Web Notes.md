## Enable Logging

  - ### php
    - `/etc/php5/apache2/php.ini` : `dispaly_errors: On`
    - run `systemctl restart apache2`
- ### Mysql
	- uncomment `general_log` in `/etc/mysql/my.cnf`
	- run `systemctl restart mysql`
	- Select number of password ``
- ## Inspect Log
	- `tail -f /var/log/mysql/mysql.log`
